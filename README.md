# yolov5_video_analysis
This project uses the powerful YOLOv5  model to analyze video streams and detect multiple objects such as people, vehicles, animals, and more in real time.  It processes video input frame-by-frame and outputs annotated video with bounding boxes, class labels, and confidence scores.
🚀 Features

🎯 Multi-class object detection (COCO classes or custom)

🎥 Supports video files and webcam streams

⚡ Real-time detection with high FPS

📦 Bounding boxes with labels & confidence

🧠 Pre-trained YOLOv5 model support

🔧 Custom dataset training capability

💾 Output video saving option
